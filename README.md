# Explore

Portal to explore Elden Ring lore and data.

Built with Angular.

## Pre-requisites

Install Node v16

`npm install`

## Run locally

`ng serve`

## Publish new version

`ng build --output-path docs --base-href "https://elden-ring-playground.github.io/explore/"`

`cp docs/index.html docs/404.html`

(git add, commit, push)
